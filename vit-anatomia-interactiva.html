Anatomía del Vision Transformer

Cada componente tiene un propósito específico. Pasa el cursor sobre cada elemento para más detalles.

🏷️

CLS Token

Classification Token

Vector aprendible añadido al inicio de la secuencia. Recopila información global de todos los patches a través de las capas de atención.

💡 Al final, solo este token se usa para la clasificación.
🧩

Patch Embeddings

Image → Sequence

La imagen se divide en patches (ej: 16×16). Cada patch se aplana y proyecta a un vector de dimensión D (ej: 768).

📐 224×224 / 16×16 = 196 patches
📍

Positional Embeddings

Spatial Information

Vectores aprendibles que se suman a cada patch para preservar la información espacial. Sin ellos, el modelo no conoce la posición.

🎯 Esenciales para entender la estructura 2D
🔗

Multi-Head Attention

Global Communication

Cada patch genera Query, Key, Value. Múltiples "cabezas" aprenden diferentes tipos de relaciones en paralelo.

Attention(Q,K,V) = softmax(QKT/√d)V
🔄

Residual Connections

Skip Connections

output = x + Sublayer(x). Mitigan el vanishing gradient y preservan información entre capas profundas.

🚀 Permiten arquitecturas de 12-32 capas
📊

Layer Normalization

Training Stability

Normaliza las activaciones por muestra. Estabiliza el entrenamiento y acelera la convergencia.

⚡ Se aplica después de cada subcapa

Encoder Block del Transformer

El bloque fundamental que se repite N veces en el Vision Transformer.

1. Multi-Head Self-Attention

Cada patch genera tres vectores: Query (Q), Key (K) y Value (V).

Attention(Q,K,V) = softmax(QKT / √dk) · V

Múltiples "cabezas" permiten aprender diferentes tipos de relaciones simultáneamente.

2. Add & Norm

Residual: output = x + Sublayer(x)
LayerNorm: Normaliza las activaciones

Las conexiones residuales permiten entrenar redes muy profundas sin vanishing gradient.

3. Feed-Forward Network

Dos capas lineales con activación GELU en medio:

FFN(x) = Linear(GELU(Linear(x)))
Dimensión: D → 4D → D

Procesa la información de atención para la siguiente capa.

💡 Concepto Clave

Este bloque se repite N veces (típicamente 12, 24 o 32). Cada capa refina las representaciones, permitiendo que la información fluya y se combine de maneras cada vez más complejas.
